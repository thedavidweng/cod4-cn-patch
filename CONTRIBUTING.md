# Contributing

Thanks for your interest in contributing.

## Getting Started

```bash
git clone https://github.com/thedavidweng/cod4-cn-patch.git
cd cod4-cn-patch
```

No dependencies required — the installer uses only Python 3 standard library.

## Development

```bash
# Test the installer (requires COD4 game files)
python3 cod4_cn_patch.py

# On Windows, double-click:
# Windows双击安装中文补丁.bat

# On macOS, double-click:
# macOS双击安装中文补丁.command
```

## Pull Requests

1. Fork the repository and create a feature branch.
2. Test on at least one platform (Windows, macOS, or Linux).
3. Open a pull request against `main`.

## Commit Messages

This project follows [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation only
- `chore:` maintenance task
